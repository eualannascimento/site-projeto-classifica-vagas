# Ordenação de seções

**Status:** Concluído  
**Data:** 2026-07-19

## Objetivo

Permitir ativar, desativar e ordenar as seções do currículo. Dados pessoais permanece incluído e fixo em primeiro lugar.

## Requisitos

- Apenas dados pessoais é obrigatório.
- As demais seções são opcionais.
- A lista permite reordenação por arrastar e soltar.
- A ordem é persistida em `enabledSections` e usada pelo wizard, prévia e impressão.
