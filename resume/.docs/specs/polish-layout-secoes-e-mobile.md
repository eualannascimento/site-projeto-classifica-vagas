# Refinamento de layout

**Status:** Concluído  
**Data:** 2026-07-19

## Objetivo

Melhorar densidade, hierarquia e ordenação das seções em desktop e mobile.

## Requisitos

- A lista de seções tem largura de leitura confortável e instrução de ordenação.
- Seções ativas e inativas possuem contraste visual distinto.
- No mobile, cada seção ativa oferece controles de subir e descer.
- A barra de prévia é auxiliar à navegação e o painel usa cabeçalho compacto.
